1. INSTALLATION
To install all the requirements, please run:
  npm install
  npx playwright install

2. RUN THE TEST
You could run the test by one of these commands:
  npm test
  npm test-ci
  npx playwright test
  npx playwright test --headed

The `npm test` and `npx playwright test --headed` are equal
The `npm test-ci` equals to `npx playwright test`
